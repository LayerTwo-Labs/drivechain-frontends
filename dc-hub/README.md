# Drivechain Hub

The per-network landing page for Drivechain deployments: sidechain overview, enforcer RPC console,
connection info, and (where deployed) a faucet for obtaining free (and worthless!) coins.

Which services are shown is driven by <https://drivechain.dev/config>; the `NETWORK` env var selects
which network entry applies to a deployment.

A running version of this is available at
[node.signet.drivechain.info](https://node.signet.drivechain.info).
